# Discord Bot Collection

A GitHub-safe version of a Python Discord bot project prepared for college submission and manual upload.

## Included apps
- Main moderation bot
- Message panel bot
- Payment bot
- Onboarding bot

## Setup
1. Install dependencies with `pip install -r requirements.txt`
2. Create a local `.env` file
3. Copy variable names from `.env.example`
4. Fill in your own local values
5. Run one app or the launcher:
   - `python main.py`
   - `python payment_bot/main.py`
   - `python msg_bot/main.py`
   - `python start_bots.py`

## GitHub upload
Upload all files from this folder.
Do not upload a real `.env` file or any local database/cache files.
